Welcome to Bootstrap Installer.

Bootstrap Installer is an auto-installer script that runs on your web server to create and publish a Bootstrap base to build off of. If you would like to initially test it out, feel free to clone the repository into your web root directory:

$ cd web_folder/www
$ mkdir bootstrap-installer
$ cd bootstrap-installer
$ git clone https://github.com/Oshizu/bootstrap-installer.git

Or you can use one of the buttons on the left to manually download or view the project.

What is Bootstrap Installer?

As mentioned above, Bootstrap Installer is an auto-installer for deploying the Bootstrap framework on your web environment. It is aimed at those who want to dive into Bootstrap with everything in place at the start, or developers who want everything to work right out-of-the-box. The installer will automatically generate PHP files ready to be used for widely-used components such as static headings across pages and MySQLi connectivity. All of this can be manually changed to fit your website's needs.

Bootstrap Installer comes shipped with a dynamic, responsive, and easy-to-use install script that will help guide you through a seamless install and configuration of Bootstrap.

Prerequisites

Bootstrap Installer's only requirement is a web server (Apache or Nginx) with PHP version 5.3+.

Optionally, you must also have a MySQLi-supported server for database connections.

How to install

To install and setup Bootstrap Installer, just download or clone the source and upload it to your web server's public folder. Then point your browser to http://yourdomain.com/install and follow the instructions.
